# 费曼技巧 (The Feynman Technique)

> "如果你不能简单地解释它，你就没有真正理解它。" —— 理查德·费曼

费曼技巧不仅仅是记忆法，更是一种**深度理解**的方法。它通过“以教代学”的方式，强迫大脑对信息进行重组、简化和输出，从而将短期记忆转化为长期理解。

## 核心流程：Concept -> Teach -> Review -> Simplify

### 1. 选择概念 (Concept)
- **预备工作**：不要对完全陌生的领域直接使用费曼技巧。请先阅读教材、观看视频，建立基本的认知基础。
- 拿出一张白纸，在顶部写下你想理解的概念或难题。
- **知识映射 (Knowledge Mapping)**：先不看书，凭记忆把你知道的关于这个概念的一切都写下来（Brain Dump），这能迅速暴露你最初的知识缺口。
- **关键点**：明确边界，不要试图一次吞下整个学科，专注于一个具体的知识点。

### 2. 试着教学 (Teach)
- 想象你面前坐着一个 **8岁的小学生**（或者一个完全不懂这个领域的门外汉）。
- 用**自己的语言**（不是书本上的定义）把这个概念讲给他听。
- **禁忌**：
  - 禁止使用专业术语（除非你先解释了它）。
  - 禁止照本宣科。
- **技巧**：
  - 多用**类比**（"这就像..."）。
  - 多用**生活中的例子**。
  - 画图辅助说明。

### 3. 回顾卡壳 (Review)
- 在解释的过程中，你会在哪里卡住？哪里解释不清楚？哪里不得不回头看书？
- **这些卡壳点就是你的盲区**。
- 回到原始材料（书、笔记、老师），重新学习这一部分，直到你能顺畅地解释它。

### 4. 简化与升华 (Simplify)
- 当你能顺畅解释后，尝试进一步简化语言。
- 能不能用更少的字说清楚？
- 能不能找到一个更精准的比喻？
- **终极目标**：把复杂的逻辑变成直觉。
- **固化成果**：将你最终打磨好的“大白话解释”制作成一张**闪卡 (Flashcard)**，正面是概念，背面是你的解释，放入间隔复习系统。

## 适用场景
- **概念抽象**：如“量子力学”、“递归”、“边际效应”。
- **逻辑复杂**：如法律条款的适用条件、复杂的数学证明。
- **本质还原**：当你觉得自己“记住了但不会用”时。

## 与其他记忆法的配合
- **前置**：先用费曼技巧把知识“吃透”。
- **后置**：理解之后，再用**林彼得记忆法**（图像/口诀）把核心关键词锁死在脑海里。
