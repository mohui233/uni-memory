# 林彼得记忆增强体系 (Universal Edition)

核心理念：将“逻辑—图像—机械”三种记忆模式协同运作，实现从“理解”到“具象”再到“稳固”的质变。

## 核心流程：解构 -> 图像 -> 记忆 -> 强化

### 1. 解构 (Deconstruction) - 逻辑层
针对复杂的定义、概念或错题，首先进行逻辑拆解。
- **5W1H 提问法**：
  - **Subject (主体)**：谁/什么东西？
  - **Action (动作)**：做什么/发生了什么？
  - **Result (结果)**：产出了什么/导致了什么？
  - **Constraint (制约)**：有什么限制条件？
- **去除冗余**：剔除修饰语，只保留核心骨架。

### 2. 图像转化 (Image Conversion) - 图像层
利用右脑视觉能力，将抽象信息转化为具象画面。
- **谐音法 (Sound)**：适用于无逻辑列表、数字、生僻词。
  - *例*：`3.14159` -> 山巅一寺一壶酒。
  - *例*：`韩赵魏楚燕齐` -> 喊赵薇出演戏。
- **象形法 (Shape)**：适用于符号、形状。
  - *例*：`W` -> 波浪；`B` -> 眼镜；`1` -> 蜡烛；`2` -> 天鹅。
- **意象法 (Meaning)**：适用于概念。
  - *例*：`递归` -> 两面相对的镜子；`自由` -> 飞翔的鸟。

### 3. 记忆系统 (Storage System) - 结构层
- **锁链法 (Link/Chain Method)**：
  - 适用：短列表、流程、步骤。
  - 原理：将图像A和图像B像锁链一样扣在一起。
  - 要诀：动词是胶水，画面越荒诞、夸张、反逻辑越好。
- **故事法 (Story Method)**：
  - 适用：长逻辑链。
  - 原理：将一连串关键词编织进一个有情节的剧本中。
- **定位系统/记忆宫殿 (Loci/Memory Palace)**：
  - 适用：庞大的知识体系、多层级信息。
  - **地点桩**：利用熟悉的物理空间（如家、办公室、上班路）存放知识点。
  - **身体桩**：利用身体部位（头、眼、鼻、口...）作为临时挂钩。
  - **数字桩**：利用数字编码（如1-100的固定图像）作为挂钩。

### 4. 高级结构模型 (Advanced Structural Models)
针对不同类型的信息，采用特定的结构模型进行处理：
- **矩阵模型 (Matrix Model)**：
  - 适用：表格、对比数据。
  - 方法：
    - **横向并联**：将每一行的数据编成一个小故事。
    - **纵向串联**：将每一列的属性（如“年份”、“事件”）分别串联。
- **金字塔模型 (Pyramid Model)**：
  - 适用：层级分明的知识体系（如生物分类、组织架构）。
  - 方法：先记忆顶层宏观概念，再利用“锁链法”逐层向下挂载微观细节。
- **地图模型 (Map Model)**：
  - 适用：地理位置、空间布局。
  - 方法：直接在真实或虚拟的地图上进行定桩，利用空间方位关系辅助记忆。

### 5. 强化 (Reinforcement) - 机械层
- **提取练习**：不看书，直接回忆（"白纸法"）。
- **间隔重复 (Spaced Repetition)**：遵循艾宾浩斯遗忘曲线。
  - *关键节点*：20分钟、1小时、1天、1周、1个月。
