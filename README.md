# 通用记忆辅助插件 (Uni-Memory)

> "把知识点截图进来，剩下的交给我。"

本插件是一个 **全领域通用** 的学习辅助工具，旨在解决“记不住、理解不透、错题反复犯”的学习痛点。

它不局限于特定学科，而是通过一套**记忆组合拳**——整合了 **费曼技巧** (理解)、**组块化记忆法** (整理) 和 **林彼得记忆法** (编码) 的底层逻辑，自动对你提供的学习材料进行“降维打击”。

## 核心功能

### 🚀 智能记忆分析 (`/auto-memory`)

这是本插件的核心命令。你只需要做一件事：**把难记的知识点、概念定义或错题截图发出来**。

插件会自动执行以下“四步走”策略，打出一套记忆组合拳：

1.  **第一拳：本质还原 (Feynman Technique)**
    - *目标*：解决“没听懂”的问题。
    - *动作*：用8岁小孩能听懂的大白话解释核心概念，多用生活类比。

2.  **第二拳：逻辑整理 (Chunking Method)**
    - *目标*：解决“太乱、太长”的问题。
    - *动作*：运用“组块化”思维，将长列表、多要点进行分类、分组，提取核心骨架 (5W1H)。

3.  **第三拳：记忆编码 (Lin Peter Method)**
    - *目标*：解决“记不住”的问题。
    - *动作*：
        - **逻辑链**：梳理因果关系。
        - **图像化**：生成谐音口诀或荒诞的“脑内剧场”故事 (锁链法/故事法)。
        - **定桩法**：必要时调用记忆宫殿或身体桩。

4.  **收招：复习建议 (Review)**
    - *目标*：解决“忘得快”的问题。
    - *动作*：基于艾宾浩斯遗忘曲线提醒你何时复习。

## 使用场景示例

- **场景 1：概念太抽象**
  - *输入*：一张关于“量子纠缠”的书本截图。
  - *输出*：
    - (费曼) 一个关于“双胞胎心灵感应”的生动比喻。
    - (林彼得) 一个辅助记忆的脑内画面。

- **场景 2：错题记不住**
  - *输入*：一张做错的英语语法题截图。
  - *输出*：
    - (费曼) 剖析为什么你会掉进这个陷阱。
    - (林彼得) 一个顺口溜帮你记住这个语法点。

- **场景 3：列表背不下来**
  - *输入*：历史朝代顺序表 / 元素周期表 / 法律条款。
  - *输出*：
    - (组块) 将长列表切分为有意义的小组。
    - (林彼得) 一套串联记忆的荒诞故事或身体桩编码。

## 知识库 (Skills)

本插件内置了三大核心记忆方法论：

- **[林彼得记忆法](skills/1_林彼得记忆法.md)** (编码)：负责将抽象信息转化为具象的图像、口诀或故事。
- **[费曼技巧](skills/2_费曼技巧.md)** (理解)：负责“以教代学”，通过简化和类比来确保深度理解。
- **[组块化记忆法](skills/3_组块化记忆法.md)** (整理)：负责突破短时记忆限制，将海量信息打包成易于管理的“组块”。
